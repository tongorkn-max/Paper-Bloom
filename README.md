# 📌 Project Overview

Paper & Bloom is a modern online stationery and gift shop created using the Wix no-code platform. The website provides customers with a simple and enjoyable shopping experience by offering a carefully selected collection of notebooks, planners, greeting cards, pens, pencils, wrapping paper, washi tape, and other stationery essentials.

The website is designed with a clean, user-friendly interface that makes it easy for visitors to browse products, explore categories, add items to the shopping cart, and contact the store. The project demonstrates the use of a no-code platform to develop a responsive and professional e-commerce website.

# ✨ Features Implemented

* Responsive homepage
* Product catalog with 21 products
* Product categories
* Product images, prices, and descriptions
* Shopping cart functionality
* Product search and filtering
* About Us page
* Mission Statement

* Mobile-responsive layout
